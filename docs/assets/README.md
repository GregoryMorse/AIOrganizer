# AIOrganizer brand assets

- `aiorganizer-mark.svg` is the primary square application/repository mark.
- `aiorganizer-logo.svg` is the horizontal GitHub/documentation wordmark.
- `aiorganizer-mark-mono.svg` is the one-color fallback for stamps and constrained output.

The runtime copy is packaged from `src/ai_organizer/resources/icons/aiorganizer.svg`. Keep the
geometry and colors aligned when revising the primary mark. SVGs contain accessible titles or labels.
