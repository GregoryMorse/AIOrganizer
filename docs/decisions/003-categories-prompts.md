# ADR-003: Categories and prompt layers

Status: accepted

Categories are user-defined hierarchical policy objects. Folder assignments
inherit and may be overridden. Privacy and destination rules are enforced in
domain validation, not delegated to models.

Prompts compile from immutable safety/schema layers followed by versioned user
workspace, view, category, action, and run guidance. Evidence is appended as
explicitly untrusted data. Proposal provenance stores all contributing revisions.
